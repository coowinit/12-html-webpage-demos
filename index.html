Guide

Use this repo on GitHub

The homepage is ready to act as a portfolio-style gallery, and each case already has its own standalone entry file under the cases/ directory.

01

Preview locally

Open the root index.html with Live Server or any static server to browse all 12 demos from a single homepage.

02

Deploy with GitHub Pages

Push the repository to GitHub, enable Pages from the main branch, and use the root folder as the publishing source.

03

Keep extending

Update the data in assets/js/main.js, replace screenshots, refine descriptions, or add new categories as your collection grows.
